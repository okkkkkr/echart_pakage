#### 相关命令

-   项目运行 `npm run dev`

-   项目打包 `npm run build`

#### 目录结构

-   api `接口管理`

-   assets `可进行编译的静态文件`

-   components `公共组件`

-   libs `JS插件库`

    -   ajax `封装原生xhr对象实现ajax请求，包含请求/响应拦截器`

    -   echarts `封装echarts插件`

    -   filters `全局过滤器`

    -   func `公共方法库`

    -   import `插件按需引用`

-   mock `模拟数据`

-   router `路由`

-   store `状态管理器`

-   views `页面`

#### 优势

-   封装布局组件

-   封装图表插件

-   封装 ajax 请求

-   封装代码自动生成脚本

-   添加暂无数据提示
